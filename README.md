# Subtractive Proteomics for Drug Target Identification

This project focuses on identifying potential therapeutic drug targets using subtractive proteomics and bioinformatics techniques. The workflow includes protein filtering, removal of human homologs, essential gene identification, and target analysis.

## Features
- Protein sequence filtering
- Essential protein identification
- Non-homologous protein analysis
- Drug target screening
- Bioinformatics data processing

## Technologies Used
- R Programming
- Bioinformatics Tools
- CSV Data Analysis
- dplyr, readr libraries

## Objective
To identify pathogen-specific proteins that can serve as potential drug targets without affecting human proteins.
